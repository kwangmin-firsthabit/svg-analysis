## Points (점)

| 이름         | 타입               | 설명                                                                                                                                                                                      | 구현 여부 |
| ------------ | ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------- |
| 좌표 기반 점 | `point`            | `x`, `y`(선택 `z`)로 직접 정의하는 점                                                                                                                                                     | ⭕        |
| 구성 기반 점 | `point.construct`  | 기하 구성으로 좌표를 렌더러가 계산 (`midpoint`, `foot`, `incenter`, `circumcenter`, `centroid`, `orthocenter`, `intersection`, `reflection`, `excenter`, `divisionPoint`, `tangentPoint`) | ⭕        |
| 경로 위 점   | `point.onPath`     | 경로 ID 또는 두 점 ID 사이를 `ratio`로 지정하는 점                                                                                                                                        | ⭕        |
| 함수 기반 점 | `point.onFunction` | 함수 ID와 x값으로 y를 자동 계산해 생성하는 점                                                                                                                                             | ⭕        |
| 점 표시 타입 | `PointType`        | `solid`, `open`, `cross`, `tick`                                                                                                                                                          | ⭕        |

## Shapes (도형)

| 이름        | 타입         | 설명                                                        | 구현 여부 |
| ----------- | ------------ | ----------------------------------------------------------- | --------- |
| 삼각형      | `triangle`   | 세 꼭짓점 기반 도형                                         | ⭕        |
| 사각형      | `rectangle`  | 네 꼭짓점 기반 도형                                         | ⭕        |
| 원          | `circle`     | `center+radius`, `center+through(1점)`, `through(3점)` 지원 | ⭕        |
| 반원        | `semicircle` | 지름 양 끝점 + `bulgeTo`                                    | ⭕        |
| 다각형      | `polygon`    | 3개 이상 꼭짓점 배열                                        | ⭕        |
| 호          | `arc`        | 중심, 반지름, 시작/끝 각도 기반                             | ⭕        |
| 부채꼴      | `sector`     | 중심점 + 호 양 끝점 기반                                    | ⭕        |
| 타원        | `oval`       | 중심, 장축 끝점, 단축 끝점 기반                             | ⭕        |
| 적분 영역   | `integral`   | 함수-축 또는 함수-함수 사이 면적                            | ⭕        |
| 부등식 영역 | `inequality` | 함수 경계/축 경계/range 모드 채움                           | ⭕        |

## Functions (함수)

| 이름          | 타입                | 설명                               | 구현 여부 |
| ------------- | ------------------- | ---------------------------------- | --------- |
| 일차함수      | `linear`            | y=ax+b                             | ⭕        |
| 이차함수      | `quadratic`         | y=ax^2+bx+c                        | ⭕        |
| 삼차함수      | `cubic`             | y=ax^3+bx^2+cx+d                   | ⭕        |
| 사차함수      | `quartic`           | y=ax^4+bx^3+cx^2+dx+e              | ⭕        |
| 지수함수      | `exponential`       | y=ab^(x-h)+k                       | ⭕        |
| 로그함수      | `logarithmic`       | y=alog_b(x-h)+k                    | ⭕        |
| 삼각함수      | `sin`, `cos`, `tan` | y=afunc(bx+c)+d                    | ⭕        |
| 절댓값 함수   | `absolute`          | y=aabs(x-h)+k                      | ⭕        |
| 유리함수      | `rational`          | y=a/(x-h)+k                        | ⭕        |
| 분기함수      | `branch`            | 조건별 `funcType`/`value`를 조합   | ⭕        |
| 일반식 함수   | `expression`        | 문자열 수식(expr) 기반 함수 그래프 | ⭕        |
| 정규분포 함수 | `normalDist`        | 평균/분산 기반 정규분포 곡선       | ⭕        |
